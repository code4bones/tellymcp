# Screenshots

This directory contains product screenshots for the current gateway-first TellyMCP model.

Relevant current flows:

- gateway Telegram menu
- browser screenshots and stored artifacts
- project collaboration
- ask/share/live flows between consoles
- Telegram Mini App / Live View

## Legacy Images

Some older images in this folder still show legacy concepts such as:

- pairing
- inbox-first wording
- local linked-session menus

Treat those as historical only. The canonical runtime model is now:

- gateway bot as the main control plane
- live consoles discovered automatically on the gateway
- collaboration through projects and `partner_note`
